Política de Privacidade

Os dados pessoais são coletados neste site apenas na medida tecnicamente necessária. Em caso algum, os dados coletados serão vendidos ou divulgados a terceiros por qualquer motivo. A informação que se segue fornece um resumo dos dados coletados neste site.
Ficheiros de log: Ao usar esta ferramenta, o site coleta os dados que o seu navegador transfere para o servidor. Isto geralmente inclui informação sobre o tipo e a versão do seu navegador, o seu sistema operativo, o URL de referência (a página Web de onde originou a sua visita), o nome do anfitrião do servidor (endereço IP), bem como o carimbo de data / hora e o pedido do servidor. Normalmente, não é possível relacionar a sua visita com uma pessoa em particular. Não comparamos os dados com outros dados que temos, nem coletamos/combinamos conjuntos de dados diferentes. Os dados são usados apenas para fins estatísticos e são eliminados regularmente.

Este site usa o Piwik, um software de código aberto para avaliação estatística e análise de visitas de páginas. O Piwik usa cookies. O cookie criado pela Piwik contém informações sobre o seu uso do site e essas informações são armazenadas nos nossos servidores na Alemanha. O endereço IP é, após processamento, diretamente anonimizado. Você pode evitar a criação de cookies através da configuração do seu navegador. Contudo, alertamos que você poderá não conseguir usar todas as funções do site. Além disso, você tem a opção de recusar qualquer processamento de dados feito pela Piwik no futuro. Basta desmarcar a caixa de seleção abaixo deste texto. Ao fazer isso, a Piwik criará um cookie de desativação da Piwik no seu navegador. Por favor note que este cookie será eliminado, se você excluir todos os cookies do seu navegador.

Aqui você pode decidir se esta ferramenta tem permissão para criar um cookie de análise da Web no seu navegador, o qual pode ajudar-nos a avaliar e analisar diferentes dados da sua visita à página. Se você é contra isto, clique no link para criar um cookie que desabilita o Piwik no seu navegador.
